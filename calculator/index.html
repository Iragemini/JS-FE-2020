• Для того, чтобы сделать число отрецательным используйте кнопку ±.
  • Возведение в степень:
    1. введите число - основание степени
    2. нажмите кнопку xy
    3. введите значение степени
  • Извлечение квадратного корня:
    1. Введите число
    2. Нажмите кнопку √
  • При выполнении недопустимой операции выводится сообщение с текстом "invalid input"
